# 整理券システム開発

このディレクトリは、光芒祭における**入場予約・整理券管理システム**の開発資料を格納します。

## 概要

光芒祭の教室展示・ステージイベント・ワークショップ等に対し、来場者が**事前予約**または**当日の整理券取得**を通じて入場枠を確保できるシステム。Airwait / Airリザーブのような整理券・予約管理機能を提供する。

- **既存POSシステムとは独立**したフロントエンド（別URL）として提供
- **アカウント基盤は共有**（POSシステムと同一のUser/Organization）
