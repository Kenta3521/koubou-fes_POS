# Phase 5: システム管理画面

**期間:** 2026年6月  
**目標:** 実行委員会が全体を管理できる

---

## 1. 概要

このフェーズでは、SYSTEM_ADMIN向けの管理機能を実装します。全団体の売上監視、団体管理、監査ログ、システム設定を完成させます。

## 4. 追加DB設計

```prisma
model SystemSetting {
  id        String   @id @default("singleton")
  eventDate DateTime?
  eventStartTime String?
  eventEndTime   String?
  systemEnabled  Boolean @default(true)
  updatedAt DateTime @updatedAt
}
```
